## How to use

add repository to composer.json

```
"repositories": [
    {
        "type": "composer",
        "url": "https://satis.bespin-studios.com"
    }
]
```

install byteShard

`composer require byteshard/rest`

## Development

Clone repository, execute ``composer update``
